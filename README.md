# 🏦 Bank System (OOP Project)

A complete banking system built using **C++** applying real-world **Object-Oriented Programming (OOP)** concepts.

---

## 🚀 Project Overview

This project simulates a real banking system where users can manage clients, perform transactions, and control access through a secure authentication system.

It was built as part of my journey to deeply understand how to design and structure real-world software systems using OOP.

---

## 🔹 Features

### 👤 Client Management

* Add new clients
* Update client information
* Delete clients
* Find clients
* List all clients

---

### 💰 Transactions System

* Deposit money
* Withdraw money
* Transfer between clients
* View total balances

---

### 👥 User Management

* Add, update, delete users
* Find users
* Manage system access

---

### 🔐 Authentication & Security

* Login & Logout system
* Permission-based access control
* Lock system after 3 failed login attempts
* Encrypted passwords storage

---

### 📊 Logging System

* Log login history
* Log transfer operations
* View logs through dedicated screens

---

### 🖥️ Console UI

* Multi-screen console interface
* Main Menu
* Transactions Menu
* Manage Users Menu
* Dynamic display of current date & logged-in user

---

## 🧠 OOP Concepts Applied

* Encapsulation
* Abstraction
* Separation of Concerns
* Modular Design
* Interface / Abstract Classes

---

## 🛠️ Technologies Used

* C++
* File Handling (Text Files as Database)
* Object-Oriented Design

---

## 📂 Project Structure

The project is structured into multiple classes representing different system components such as:

* Clients

* Transactions
* Screens (UI)
* Utilities
